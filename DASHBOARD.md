## Directory Layout Explanation

- **Root**: Contains main configuration, orchestration scripts (`scripts/`), universal documentation (`docs/`), and the workspace-level manifest files (`package.json`, `pyproject.toml`, `.gitmodules`).
- **AI & Orchestration**: `borg/`, `mcp-superassistant/`, `antigravity-autopilot/`, `jules-autopilot/`, `opencode-autopilot/`, `.agent/` — Multi-modal AI agents and autonomous development loops.
- **Rhythm Games** (`bobmani/`): `bobmania/`, `itgmania/`, `beatoraja/`, `arrowvortex/`, `ddc/`, `ddc_onset/`, `ksm-v2/`, `linthesia/`, `pianogame/`, `Simply-Love-SM5/`, `hymnmania/`, `leraine-studio/`, `ffr-difficulty-model/` — Comprehensive rhythm gaming ecosystem.
- **Bob Ecosystem**: `bobfilez/`, `bobsaver/`, `bobeditpro/`, `bobtorrent/`, `bobui/`, `bobium/`, `bobzilla/`, `bobtrax/`, `bobtrader/`, `bobcoin/`, `bobdesk/`, `bobbybookmarks/`, `bobzzite/` — Personal productivity and desktop application suite.
- **Games & Engines**: `bg/`, `mk64/`, `sm64coopdx/`, `neverball/`, `superpowers/` — Game engines, decompilation projects, and game development tools.
- **Web Applications**: `fwber/`, `raindropioapp/` — Full-stack web applications.
- **Developer Tools**: `topaz-ffmpeg/`, `dupeguru/`, `picard/`, `frontend-sdl-cpp/`, `OmniRoute/` — Specialized utilities and libraries.
