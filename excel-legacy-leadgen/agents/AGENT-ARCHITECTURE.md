# AGENT & SUBAGENT ARCHITECTURE
# The Excel Legacy Realty Group Lead Generation System

## HOW THIS WORKS
Each "agent" is a specialized AI prompt/workflow that handles one domain.
The Master Orchestrator coordinates all agents. Subagents handle specific
tasks within each agent's domain. This creates parallel execution -- we
run 14+ platform strategies simultaneously without bottlenecking.

---

## MASTER ORCHESTRATOR
Role: Coordinates all agents, tracks progress, allocates resources.
Location: agents/orchestrator.md

## AGENT ROSTER

### AGENT 1: Content Director
Role: Creates ALL content across all platforms.
Subagents:
  - 1A: Listing Content Writer (property descriptions, listing posts)
  - 1B: Market Analyst (reports, data, statistics)
  - 1C: Community Content Creator (local events, neighborhood guides)
  - 1D: Video Script Writer (YouTube, TikTok, Reels scripts)

### AGENT 2: Social Media Manager
Role: Schedules, posts, engages on all social platforms.
Subagents:
  - 2A: Facebook Specialist (posts, groups, events, lives)
  - 2B: Instagram Specialist (feed, stories, reels)
  - 2C: TikTok Specialist (short-form video, trends)
  - 2D: X/Twitter Specialist (posts, threads, engagement)
  - 2E: LinkedIn Specialist (professional content, networking)
  - 2F: Pinterest Specialist (boards, pins, SEO)
  - 2G: YouTube Specialist (long-form, shorts, community)
  - 2H: Nextdoor Specialist (hyperlocal posts, recommendations)

### AGENT 3: Lead Generator
Role: Identifies and captures leads from all sources.
Subagents:
  - 3A: Inbound Lead Manager (captures, qualifies, routes leads)
  - 3B: FSBO Outreach Agent (finds and contacts FSBO sellers)
  - 3C: Expired Listing Agent (follows up on expired listings)
  - 3D: Circle Prospecting Agent (targets around new listings/sales)
  - 3E: Probate/Estate Lead Agent (courthouse record monitoring)
  - 3F: Social DM Agent (engages prospects via social DMs)

### AGENT 4: SEO & Local Search
Role: Dominates Google search and local pack results.
Subagents:
  - 4A: Google Business Profile Manager (posts, Q&A, reviews)
  - 4B: Local SEO Writer (city pages, neighborhood content)
  - 4C: Review Generation Agent (solicits and responds to reviews)
  - 4D: Citation Builder (directory listings, NAP consistency)

### AGENT 5: Paid Advertising
Role: Manages paid campaigns across all platforms.
Subagents:
  - 5A: Facebook/IG Ads Manager (lead forms, retargeting)
  - 5B: Google Ads Manager (search, display, YouTube)
  - 5C: Zillow/Realtor.com Manager (premier agent profiles)
  - 5D: Campaign Analyst (ROAS, CPL, optimization)

### AGENT 6: Brand & Reputation
Role: Builds and protects the Excel Legacy brand.
Subagents:
  - 6A: Brand Voice Guardian (ensures consistency)
  - 6B: Review Responder (responds to all reviews)
  - 6C: PR & Media Agent (press releases, local media)
  - 6D: Partnership Agent (lender, inspector, contractor partners)

### AGENT 7: Outreach & Drip
Role: Manages all proactive outreach and nurture sequences.
Subagents:
  - 7A: Email Drip Writer (seller drip, buyer drip, sphere)
  - 7B: SMS Outreach Writer (text scripts for leads)
  - 7C: Direct Mail Creator (postcards, letters, flyers)
  - 7D: Cold Call Script Writer (phone scripts for all scenarios)
