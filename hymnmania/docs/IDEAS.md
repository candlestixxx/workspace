# Ideas & Future Expansions

- **VR Psytrance Temple**: A 3D environment generated via AI (e.g., Skybox AI) where the hymn-psytrance tracks play with spatial audio.
- **Real-time Algorithmic Performance**: A live-coding interface where users can tweak the psytrance generation parameters (Euclidean density, Markov transition probabilities) in real-time.
- **Multi-lingual Hymn Support**: Use AI to translate ancient hymns into modern languages and generate vocals in those languages via ElevenLabs.
- **Collaboration Mode**: Allow multiple users to contribute to the same "Hymn Remix" project via a shared Web UI.
- **NFT Integration**: Automatically mint unique hymn remixes as NFTs on various blockchains.
- **Mobile App**: A "Hymn-to-Psy" app where users can hum a melody and have the pipeline generate a full track.
- **Interactive Lyric Visualizers**: Beyond simple wave visualizers, use LLMs to generate per-sentence visual prompts for tools like Stable Video Diffusion.
