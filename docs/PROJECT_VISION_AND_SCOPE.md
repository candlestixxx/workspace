# Project Vision & Scope

## Core Philosophy
We are maintaining two distinct forks of StepMania with specific goals. Ideally, these will eventually merge, but for now, they serve different purposes.

### 1. ITGMania (The "Stable" Fork)
*   **Target Audience:** Arcade cabinets, tournaments, competitive play.
*   **Base:** StepMania 3.95 (The definitive version).
*   **Key Features:**
    *   **Stability:** Rock-solid performance. No crashes allowed.
    *   **Mod Support:** Full `NotITG` Lua/Mod support.
    *   **Tournament Ready:** Built-in tournament ladders and features.
    *   **Legacy Compatibility:** 100% compatibility with older ITG content.

### 2. StepMania (The "Experimental" Fork)
*   **Target Audience:** Modern home users, online players, feature testers.
*   **Base:** StepMania 5.2+ (Modern architecture).
*   **Key Features:**
    *   **Modernization:** Features from Etterna, Outfox, and SMOnline.
    *   **Online:** Global ranking systems.
    *   **Parity:** Feature parity with StepManiaX and DDR A.
    *   **Experimental:** Playground for new mechanics and code.

## Shared Goals
Both projects should be bundled with:
*   **ArrowVortex:** For chart creation/editing.
*   **DDC (Deep Dance Chainer):** For automated chart generation.
*   **FFR Difficulty Calculator:** For accurate difficulty ratings.

## Long Term Goal
Eventually merge these forks into a single, unified "Definitive StepMania" that offers the stability of 3.95 with the features of 5.2+, likely via a strictly managed legacy mode or engine rewrite.
